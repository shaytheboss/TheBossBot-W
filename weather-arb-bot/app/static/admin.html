Performance by confidence band

Each row = positions whose confidence score falls within that band only (not cumulative). Empty bands are hidden. Model accuracy

Each weather model's point forecast scored against the bucket Polymarket settled, over resolved markets in the current filter. Helps spot which model performs best — and where (per-city), so models could later be weighted per city.

⚖️ Model skill DB (live prediction weights)

The managed, continuously-updated per-city model accuracy database. Each row scores a model's last same-day forecast against the bucket Polymarket actually settled, over a rolling 90-day window. The Weight column is what both the daily and the ⚡ intraday predictors actually use (1.0 = neutral; models below the sample minimum stay neutral until they earn a track record). Updates automatically after every settlement and hourly.
